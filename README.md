## Projeto-Calculadora
A ideia principal foi criar um dispositivo para a realização de cálculos numéricos. Com o objetivo de ser simples e de fácil compreendimento.

<br>

## Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:
- HTML5
- CSS3
